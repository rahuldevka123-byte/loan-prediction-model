
# Loan Prediction Model

A machine learning model that predicts whether a bank will approve a loan application based on applicant details and credit history.

## What it does
- Takes applicant details as input (income, education, dependents etc.)
- Checks CIBIL score based on loan type (Home, Car, Personal, Education)
- Predicts loan approval or rejection using Logistic Regression
- Achieved 80.3% accuracy on test data

## Features
- Supports 4 loan types with different CIBIL requirements
- Warns applicants with multiple existing loans
- Built using Python, Pandas, NumPy, and Scikit-learn

## Dataset
Kaggle Loan Prediction Dataset

## How to run
Open the notebook in Google Colab and run all cells
